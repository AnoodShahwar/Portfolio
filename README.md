## 🕹️ What Is This?

A fully GTA-themed interactive developer portfolio because a plain white resume just wasn't going to cut it. Built entirely in Flutter Web, designed in Figma, and deployed on Vercel. Every screen is a mission. Every project is a wanted poster. Welcome to the city.

---

## 🗺️ MISSIONS (Screens)

| ⭐ Screen | 📋 Description |
|-----------|---------------|
| 🎬 **Splash** | GTA-style opening with custom Pricedown & Diploma fonts and a gold PRESS START button |
| 👤 **About Me** | Player card with tabs; Player, Mission, Attributes (skill tree), Misc |
| 🗂️ **Projects** | Flutter Division & Unity Division; each project has a status badge (Deployed, Ongoing, Coming Soon) |
| 🛠️ **Skills** | Interactive wheeler chart + proficiency bars across Languages, Frameworks, Backend & Tools |
| 📜 **Certifications** | GTA-style folder system: Coursera, Cisco & Competitions |
| 📞 **Contact** | Initialize Connection screen: Gmail, GitHub, LinkedIn & Resume |
